Using two libraries: Lottie & ScrollMagic.

Debug tools from ScrollMagic are activated in this preview. You can remove the debugging tools from the JS.

HTML (with all attributes) is:

<div class="lottie-animation" data-anim-path="https://assets4.lottiefiles.com/packages/lf20_tpXwrC.json" data-anim-duration="" data-anim-offset="" data-anim-trigger-hook=""></div>

.lottie-animation elements has a total of three extra attributes to keep in mind:

data-anim-path [REQUIRED]
Path to the Bodymovin JSON file that this element will load.
data-anim-duration

How long is the animation? Measured in distance from trigger-start.

Can be passed:
"100%" (A percentage of the viewport - passed as string),
100 (Set number of pixels - passed as number or string),
"elemHeight" (A string - gets height of SVG),
"parentHeight" (A string - gets height of container). Defaults to the height of the parent element after the SVG has loaded if empty/not mentioned.

See ScrollMagic docs on Scene constructors.

data-anim-trigger-hook

At what point of the viewport should the trigger be registered?

Can be passed a float number [0.0 - 1.0] as a string. Defaults to vertical middle of screen - 0.5 if empty/not mentioned.

See ScrollMagic docs on Scene constructors.

data-anim-offseet
Number of pixels AFTER the trigger-start to start animation. Can be passed a number as a string. Defaults to no offset if empty/not mentioned.
See ScrollMagic docs on Scene constructors.
